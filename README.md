## static
- 类中static成员变量。当你在类中声明静态成员变量时，它只是一个声明，告诉编译器这个类有这个静态成员变量，但并没有为其分配内存空间。因此，你需要在类外的某个地方（通常是在源文件中）对静态成员变量进行定义和初始化，以为其分配存储空间。
